# 产品需求规格 (Requirements)

## 项目概述

**产品名称**：粤语学习助手（Cantonese Learner）
**平台**：iPhone（PWA 网页 App，通过 Safari "添加到主屏幕" 使用）
**核心价值**：将普通话视频转为带粤语翻译 + 拼音 + 朗读的学习材料

## 目标用户

- 想通过真实视频内容学习粤语的普通话母语者
- 对市面现有粤语学习 App 功能不满意的自学者
- 已具备一定粤语听力基础，需要字幕对照辅助的学习者

## 功能需求

### F1. 视频导入

| ID | 需求 | 优先级 |
|----|------|--------|
| F1.1 | 支持从手机相册/文件选择本地视频（.mp4 / .mov / .mkv） | P0 |
| F1.2 | 支持粘贴在线视频链接（B站、YouTube、抖音等） | P1 |
| F1.3 | 自动识别链接类型并以对应方式加载视频 | P1 |
| F1.4 | 本地视频用原生 `<video>` 播放；在线视频用 iframe 内嵌播放 | P1 |

### F2. 语音识别 (ASR)

| ID | 需求 | 优先级 |
|----|------|--------|
| F2.1 | 从本地视频提取音频（ffmpeg.wasm） | P0 |
| F2.2 | 音频发送腾讯云 ASR 进行语音识别 | P0 |
| F2.3 | 返回带时间戳的普通话逐句文本 | P0 |
| F2.4 | 无字幕视频自动识别；有字幕视频可选提取/识别 | P0 |

### F3. 翻译 + 拼音

| ID | 需求 | 优先级 |
|----|------|--------|
| F3.1 | 将普通话文本逐句翻译成地道粤语口语 | P0 |
| F3.2 | 为每句粤语标注粵拼（Jyutping）拼音 | P0 |
| F3.3 | 翻译结果与原文时间戳对齐 | P0 |
| F3.4 | 支持批量翻译（一次发送多句） | P1 |

### F4. 学习界面

| ID | 需求 | 优先级 |
|----|------|--------|
| F4.1 | 视频播放区域（约占屏幕 35%） | P0 |
| F4.2 | 三行对照字幕：普通话 / 粤语 / 拼音 | P0 |
| F4.3 | 字幕随视频播放自动滚动并高亮当前句 | P0 |
| F4.4 | 手动滑动浏览时不高亮跳回，松手 2 秒后恢复自动跟随 | P0 |
| F4.5 | 底部固定音频播放器 | P0 |

### F5. 音频播放

| ID | 需求 | 优先级 |
|----|------|--------|
| F5.1 | 整段播放模式：连续朗读所有粤语句子 | P0 |
| F5.2 | 逐句点播模式：点击字幕句子朗读该句 | P0 |
| F5.3 | 播放速度调节（0.5x / 0.75x / 1x / 1.25x / 1.5x） | P1 |
| F5.4 | 播放进度条 | P1 |
| F5.5 | TTS 音频缓存（同句不重复请求） | P1 |

### F6. API 设置

| ID | 需求 | 优先级 |
|----|------|--------|
| F6.1 | 输入 DeepSeek API Key | P0 |
| F6.2 | 输入腾讯云 ASR SecretId + SecretKey | P0 |
| F6.3 | 输入 Azure TTS Subscription Key + Region | P0 |
| F6.4 | "测试连接" 按钮分别验证各服务 | P1 |
| F6.5 | API Key 经 Web Crypto API 加密存储于 IndexedDB | P0 |
| F6.6 | 隐私声明：所有 Key 仅存本地 | P0 |

### F7. 学习记录

| ID | 需求 | 优先级 |
|----|------|--------|
| F7.1 | 历史记录列表（视频标题、导入时间、句子数） | P0 |
| F7.2 | 点击记录恢复学习会话（字幕 + 视频 + 进度） | P0 |
| F7.3 | 删除历史记录 | P1 |
| F7.4 | 保存上次播放位置 | P1 |

## 非功能需求

| ID | 需求 | 指标 |
|----|------|------|
| NF1 | 安装到 iPhone 主屏幕后以独立 App 形态运行 | PWA standalone |
| NF2 | 移动端适配 | iPhone SE ~ Pro Max |
| NF3 | 首次加载时间（含 ffmpeg.wasm） | < 30s（Wi-Fi） |
| NF4 | ASR 处理大文件（>5MB）超时 | 2 分钟 |
| NF5 | 离线可访问已缓存数据 | Service Worker |
| NF6 | 隐私安全 | 零服务端、全本地处理 |

## 不包含的功能（现阶段）

- 用户注册/登录
- 云端同步
- Android 原生适配（PWA 理论上支持，但不专门适配）
- App Store 上架
- 视频下载/离线保存（涉及版权）
