# UI/UX 设计规范 (Design Spec)

## 设计原则

- **移动优先**：iPhone 为主要目标设备（375pt ~ 430pt 宽）
- **简洁克制**：减少认知负荷，聚焦学习内容
- **触摸友好**：所有可交互元素 ≥ 44pt 点击区域
- **安全区域**：适配刘海屏 / 灵动岛 / Home Indicator

## 排版

| 层级 | 字号 | 行高 | 字重 | 用途 |
|------|------|------|------|------|
| H1 | 24px | 1.3 | Bold (700) | 页面标题 |
| H2 | 18px | 1.4 | Semibold (600) | 区块标题 |
| Body | 16px | 1.5 | Regular (400) | 正文（普通话、粤语） |
| Caption | 13px | 1.4 | Regular (400) | 拼音、时间戳 |
| Small | 12px | 1.3 | Regular (400) | 辅助信息 |

**字体家族**：
- 系统字体栈（中文优化）：`-apple-system, BlinkMacSystemFont, 'PingFang SC', 'Microsoft YaHei', sans-serif`
- 拼音可使用等宽变体或专门字体（后续优化）

## 布局规范

### 全局布局
- 全屏 Flex 纵向布局
- 页头固定在顶部（含安全区域 padding）
- 底部导航/播放器固定（含安全区域 padding）
- 内容区 flex-1 填充剩余空间

### 间距系统
- 页面左右 padding: 20px (5)
- 区块间距: 16px (4)
- 区块内间距: 12px (3)
- 卡片圆角: 16px (rounded-2xl)
- 按钮圆角: 12px (rounded-xl)

## 交互规范

### 触摸反馈
- 按钮按下：`active:scale-[0.98]` 轻微缩放
- 列表项点击：`active:bg-slate-100` 背景变色
- Transition：`transition-transform` 或 `transition-colors`
- 禁用 iOS 长按选择：`-webkit-touch-callout: none`

### 加载状态
- 首次加载：全屏 skeleton
- API 处理中：顶部进度条 + 步骤说明文字
- 音频加载：播放按钮显示 spinner

### 错误处理
- 网络错误：Toast 提示 + 重试按钮
- API 错误：具体错误信息展示
- 文件格式不支持：Alert 弹窗说明

### 空状态
- 无历史记录：居中灰色文字"暂无学习记录"
- 无字幕数据：居中提示 + 导入按钮

## PWA 规范
- **显示模式**：standalone（无浏览器 UI）
- **方向**：portrait（仅竖屏）
- **主题色**：#1e40af（与 Header 一致）
- **背景色**：#f8fafc
- **启动图**：使用 SVG 图标（粤字 logo）
