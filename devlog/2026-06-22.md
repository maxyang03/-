## 遇到的问题
- ffmpeg.wasm 需要 COOP/COEP 响应头来启用 SharedArrayBuffer，Vite dev server 通过 `server.headers` 配置，生产环境需在部署服务器设置
- Vite 8.x npx 需要在项目目录执行，否则找不到 index.html
- ffmpeg.wasm core 文件约 30MB，首次加载较慢（Wi-Fi 约 5-15 秒）
- iOS Safari 对 SharedArrayBuffer 的支持从 15.2+ 开始，条件较为严格
